What it is

Computer use is a capability Anthropic added to Claude 3.5 Sonnet (the upgraded “new Sonnet”), announced in October 2024 as a public beta through the Anthropic API. Rather than restricting the model to text or app-specific tool calls, it lets Claude perceive a screen via screenshots and act on it by moving the cursor, clicking, and typing — operating arbitrary desktop and browser GUIs through the same interface a person uses.
